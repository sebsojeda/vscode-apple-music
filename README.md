# Apple Music Player - Visual Studio Code

Apple Music client for Visual Studio Code.

## Features

- Supported Controls
  - Open Apple Music
  - Play/Pause
  - Mute/Unmute
  - Play Previous/Next Song

![screen shot](images/screen-shot.png)

## Requirements

- macOS with Apple Music
